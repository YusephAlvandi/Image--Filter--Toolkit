Image Filter Toolkit

A real-time image processing desktop app with live preview. Capable of applying edge detection, grayscale, invert, sharpen, and blur filters instantly and saving the results.

Built with Python, OpenCV, and CustomTkinter.

Run this command to install dependencies
pip install opencv-python customtkinter pillow numpy

Appropriate for offline working.

The advantages over online tools: no uploads, no privacy concerns, immediate feedback with adjustable settings.
